Cloud Computing & DevOps Fundamentals

5-Day Course Syllabus (AWS-Focused)

Day 1: Cloud Foundations & AWS Introduction

Session 1 – Cloud Fundamentals

  • Evolution of computing (Traditional vs Cloud)
  • Cloud service models (IaaS, PaaS, SaaS)
  • Deployment models (Public, Private, Hybrid)
  • Benefits of cloud (scalability, elasticity, cost-efficiency)

Session 2 – Cloud Ecosystem & Providers

  • Overview of major cloud providers
  • Introduction to AWS
  • Real-world use cases

Session 3 – AWS Basics & IAM

  • AWS Global Infrastructure (Regions, Availability Zones)
  • AWS Management Console walkthrough
  • Identity and Access Management (IAM)

Day 2: Core AWS Services & Database Fundamentals

Session 1 – Compute Services (EC2)

  • Introduction to Amazon EC2
  • Instance types
  • Key pairs and security groups

Session 2 – Storage Services (S3)

  • Amazon S3 overview
  • Storage types and use cases
  • Basic S3 usage

Session 3 – Database Fundamentals

  • What is a database?
  • SQL vs NoSQL
  • Amazon RDS and DynamoDB
  • Choosing the right database

Day 3: Networking (VPC) & DevOps Introduction

Session 1 – Networking Basics

  • Fundamentals of networking (IP, subnets)
  • Introduction to Amazon VPC

Session 2 – VPC Components & Security

  • Public vs Private subnets
  • Internet Gateway
  • Security Groups vs Network ACLs

Session 3 – DevOps Fundamentals

  • What is DevOps?
  • Traditional vs DevOps workflow
  • DevOps lifecycle
  • Automation and collaboration

Day 4: Version Control & CI/CD Pipelines

Session 1 – Version Control (Git & GitHub)

  • Introduction to Git and GitHub
  • Basic commands (clone, commit, push)
  • Branching and collaboration

Session 2 – Modern DevOps Concepts

  • Containerization basics
  • Orchestration basics

Session 3 – CI/CD Fundamentals

  • Continuous Integration vs Continuous Deployment
  • Pipeline stages (build, test, deploy)
  • Tools: GitHub Actions, Jenkins

Day 5: Monitoring, Cost Optimization & Mini Project

Session 1 – Monitoring & Logging

  • Importance of monitoring in cloud
  • Amazon CloudWatch
  • Metrics vs Logs

Session 2 – Advanced AWS Services & Cost Optimization

  • AWS Lambda (serverless concept)
  • Amazon SNS (notifications)
  • Amazon CloudFront (CDN basics)
  • AWS pricing model (pay-as-you-go)
  • Cost optimization best practices

Session 3 – Mini Project

  • Deploy a simple application on AWS
  • Use EC2/S3 with CI/CD
  • Presentation and feedback
